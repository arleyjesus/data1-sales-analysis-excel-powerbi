# data1-sales-analysis-excel-powerbi

# 📊 Análisis de Datos de Ventas con Excel y Power BI

## Descripción

Proyecto de análisis de datos basado en un conjunto de datos de ventas simulado.  
El proyecto incluye el proceso de limpieza de datos, validación de información y visualización mediante un dashboard.

---

## Dataset

El dataset contiene registros de ventas con las siguientes variables:

- Fecha
- Ciudad
- Producto
- Categoría
- Canal de venta
- Cantidad
- Precio unitario
- Ventas

El dataset inicial contiene distintos problemas de calidad de datos como:

- valores faltantes
- errores tipográficos
- inconsistencias en categorías
- registros duplicados
- cálculos incorrectos en ventas

Estos problemas fueron corregidos durante el proceso de limpieza.

---

## Limpieza de Datos

La limpieza del dataset se realizó utilizando **Excel** e incluyó:

- detección de valores vacíos
- corrección de nombres inconsistentes
- eliminación de duplicados
- corrección de errores de cálculo
- estandarización de texto

---

## Visualización

Una vez limpio el dataset, los datos se utilizaron para construir un dashboard en **Power BI** que permite analizar:

- ventas por ciudad
- ventas por producto
- ventas por canal de venta
- evolución de ventas en el tiempo
